# Plan for Fullstack-prosjekt – Restaurant Tangen Torv

## Konsept
Jeg skal lage en nettside for restauranten "Tangen Torv". Kunder skal kunne:
- Se meny
- Se åpningstider
- Finne kontaktinformasjon
- Bestille bord

Ansatte skal kunne:
- Se bestillinger
- Administrere (slette/endre) bestillinger

## Teknologier
- Database: PostgreSQL i Docker
- Backend: Node.js + Express (senere)
- Frontend: HTML, CSS, JavaScript
- Versjonskontroll: GitHub

## Grov tidsplan
Dag 1–2: Database (Modul 1)  
Dag 3–5: Nettsider med statisk data (Modul 2)  
Senere: Koble backend (Modul 3)
