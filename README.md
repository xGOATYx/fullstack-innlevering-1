# Restaurant Tangen Torv – Fullstack prosjekt

## Beskrivelse
Dette prosjektet er en fullstack-løsning for en restaurant hvor kunder kan se meny og bestille bord, og ansatte kan administrere bestillinger.

## Teknologier
- HTML, CSS, JavaScript
- PostgreSQL
- Docker
- (Senere) Node.js + Express

## Hvordan kjøre databasen
1. Installer Docker
2. Kjør:
   docker-compose up -d

## Prosjektstruktur
Forklar mappene

## Kjente feil / mangler
- Ingen ekte bestilling enda (kommer i backend)

## Videre utvikling
- Koble nettsidene til backend
- Innlogging for ansatte
